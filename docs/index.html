NEUPL syntax

NEUPL is a stack-based language. Most operations rely on the internal stack in some way.

Please note that this language is not Turing-Complete, or even anywhere close.

Commands

NEUPL recognises a few commands: 

  • <
    : Pushes the preceeding character onto the stack.
  • p
    : Pop a character, and...
    • p
      : print it.
    • i
      : insert it into the program.
  • l
    : Loop through the stack, popping each character and...
    • p
      : printing it.
    • e
      : executing it as an instruction.
